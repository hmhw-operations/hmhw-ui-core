# Accordion

Accordion component with different props.

#### Example

<Canvas of={Accordion.Default} />

## Usage

```ts
import {Accordion} from "sld-ui";

const Example = () => {
  return (
      <Accordion
        text={"Accordion"}
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`() => void`_ - A string that represents the text content of the Accordion.
